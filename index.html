Research Publication

Fly-ABAC: Attribute-Based Access Control for UAV Navigation

IEEE Big Data Conference 2024
Fly-ABAC is a novel application of Attribute-Based Access Control (ABAC) for UAV navigation which enhances security and compliance for UAV operations. The system models both positive and negative attributes and exclusion zones to enforce property access rules which gives options for property owners to enforce access. Ensures UAVs avoid unauthorized zones while adhering to property-specific access requirements. Provides granular and customizable control suitable for various scenarios, such as restricted government buildings or private properties.
